
## 이번 독립 호출의 지정 입력

- 회차: F1
- 대상 문항: **문 {Q} 하나만**
- `01_sealed/_parts/F1_voted.json`에서 문 {Q} 항목만 읽고 갱신하라.
- 현재 작업 디렉터리는 병렬 실행 시 덮어쓰기를 막기 위해 만든 동일 회차의 격리 복제본이다. 이 로컬 `F1_voted.json`을 패치하되, 다른 문항은 어떤 필드도 변경하지 마라.
- 최종 저장 전에 대상 문항에 `"deepdive": true`가 있고 answer가 1~5인지 확인하라.
- 실제 사용 모델을 대상 문항의 `deepdive_model` 필드에 정확히 자기보고하라. 기대값은 `gpt-5.6-sol`이다.
